Mortgage Loan Default - Data Preprocessing Using Machine Learning Techniques

Data preprocessing is crucial in developing a default prediction model, as it ensures the accuracy and reliability of the model by handling missing values, outliers, and noisy data, allowing for better feature extraction and improved predictive performance.

This dataset contains nearly 150,000 mortgage loan applications in 2019 with 32 attributes. Using statistical analysis and data visualization, I analyzed the data to detect outliers and missing values, as well as implemented appropriate methods of data transformation to ensure data quality for modeling purposes.
